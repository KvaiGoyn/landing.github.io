# Настройка Getform.io для обработки форм

## Обзор

Все формы на лендинге «Стиль Мастер» были переведены на использование сервиса [Getform.io](https://getform.io) — бесплатного, CORS‑совместимого сервиса для сбора заявок. Формы отправляются напрямую на эндпоинт Getform.io, что гарантирует надёжность и отсутствие проблем с CORS.

## Шаги настройки

### 1. Создание аккаунта на Getform.io

1. Перейдите на [https://getform.io](https://getform.io) и зарегистрируйтесь (можно через GitHub, Google или email).
2. После регистрации вы попадёте в панель управления, где сможете создавать формы.

### 2. Создание пяти форм (по одной для каждого типа заявки)

Getform.io позволяет создавать неограниченное количество форм на бесплатном тарифе (до 50 отправок в месяц).

Создайте следующие формы:

| Тип формы | Предлагаемое название | Эндпоинт (будет выдан после создания) |
|-----------|----------------------|----------------------------------------|
| Обратный звонок | `callback-form` | `https://getform.io/f/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |
| Контактная форма | `contact-form` | `https://getform.io/f/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |
| Консультация | `consultation-form` | `https://getform.io/f/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |
| Расчёт стоимости | `calculation-form` | `https://getform.io/f/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |
| Вызов замерщика | `measurement-form` | `https://getform.io/f/xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx` |

**Как создать форму:**
- Нажмите **«New Form»** в панели Getform.
- Введите название формы (например, `callback-form`).
- Скопируйте **Form Endpoint** (URL вида `https://getform.io/f/...`). Этот URL нужно будет подставить в соответствующий компонент.

### 3. Обновление компонентов форм

В проекте пять файлов компонентов, каждый из которых содержит форму с заглушкой `your-form-id`. Замените этот ID на реальный Form Endpoint из Getform.io.

| Компонент | Файл | Строка для замены |
|-----------|------|-------------------|
| `CallbackForm` | `app/components/forms/CallbackForm.tsx` | `action="https://getform.io/f/your-form-id"` |
| `ContactForm` | `app/components/forms/ContactForm.tsx` | `action="https://getform.io/f/your-form-id"` |
| `ConsultationForm` | `app/components/forms/ConsultationForm.tsx` | `action="https://getform.io/f/your-form-id"` |
| `CalculationForm` | `app/components/forms/CalculationForm.tsx` | `action="https://getform.io/f/your-form-id"` |
| `MeasurementForm` | `app/components/forms/MeasurementForm.tsx` | `action="https://getform.io/f/your-form-id"` |

**Пример замены для CallbackForm:**
```tsx
// Было
action="https://getform.io/f/your-form-id"

// Стало (используйте ваш реальный endpoint)
action="https://getform.io/f/abc123def456"
```

### 4. Настройка уведомлений и редиректов

В панели Getform.io для каждой формы можно настроить:

- **Email‑уведомления:** Укажите email, на который будут приходить новые заявки.
- **Редирект после отправки:** Параметр `_next` уже присутствует в скрытых полях форм (например, `value="https://yourdomain.com/thanks"`). Замените `https://yourdomain.com/thanks` на реальный URL вашей страницы «спасибо».
- **Тема письма:** Используется скрытое поле `_subject` (уже настроено для каждой формы).

### 5. Проверка работы форм

1. После замены ID запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
2. Откройте любую форму в браузере, заполните поля и отправьте.
3. Проверьте, что:
   - После отправки происходит редирект на указанную страницу (или отображается стандартная страница Getform.io).
   - В панели Getform.io появилась новая заявка.
   - На указанный email пришло уведомление (если настроено).

### 6. Дополнительные возможности Getform.io

- **Валидация и антиспам:** В настройках формы можно включить Google reCAPTCHA, Honeypot и другие фильтры.
- **Интеграции:** Getform.io поддерживает отправку данных в Google Sheets, Slack, Webhook и др.
- **Логирование:** Все отправки сохраняются в панели, их можно экспортировать.

## Частые вопросы

**Q: Нужно ли настраивать CORS?**  
A: Нет, Getform.io полностью CORS‑совместим, обычные HTML‑формы отправляются без ограничений.

**Q: Сколько форм можно отправлять бесплатно?**  
A: Бесплатный план включает 50 отправок в месяц. Для большинства лендингов этого достаточно.

**Q: Что делать, если форма не отправляется?**  
A: Проверьте:
   - Правильность Form Endpoint в атрибуте `action`.
   - Наличие обязательных полей (помечены `required`).
   - Сеть разработчика (вкладка Network) на наличие ошибок 4xx/5xx.

**Q: Можно ли использовать одну форму для всех типов заявок?**  
A: Технически можно, но для удобства фильтрации рекомендуется создать отдельные формы.

## Резервное решение

Если Getform.io по какой‑либо причине перестанет работать, вы можете:
1. Вернуться на FormSubmit.co (заменив URL обратно).
2. Использовать собственный API‑маршрут (`/api/formsubmit`), который уже реализован в проекте.
3. Перейти на другой сервис (например, Formspree, когда он снова станет доступен).
